## Changelog Judie V2.5

**Uploaded 3. February 2025**

Judie v2.5 is finally here! After a long time of no news I'm proud to announce that Judie is finally getting a content update! :ChibiAnnieYay: 
Apologies for the lack of activity working on her the last 2 years, life got pretty busy with uni and now work, but I finally found the time to at least do this.

Also with this announcement I'd like to say I'll be working on cleaning up Judie, both the code and repositories (the public repo hasn't been updated since like 2022 :lurkk:), as well as QOL improvements in the future! As always with no set timeline in mind, I'll work on it when I have both the time and motivation to do so.

Furthermore I am now opening up my judie-related discord server so you guys can contribute to Judie! That's right, in the forums there you'll be able to submit suggestions to almost all things Judie! From quotes to pictures all the way to directly suggesting features, the stage will be yours! 
Invite link: https://discord.gg/usCwNK6ruH
